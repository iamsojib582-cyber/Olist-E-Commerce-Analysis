# 📊 Olist E-Commerce Analytics (Data Analyst • Data Engineering • Power BI)

An end-to-end analytics project built on the Brazilian Olist e-commerce dataset. This project demonstrates the full data workflow — from raw data processing to business insights and interactive dashboards.

---

## 🚀 Project Overview

This project integrates **data engineering, SQL analytics, and BI visualization** to analyze sales performance, customer behavior, and delivery efficiency.

---

## 🧠 Roles & Responsibilities

### 🔧 Data Engineering

* Cleaned and transformed raw CSV datasets using Python
* Handled missing values, duplicates, and inconsistent formats
* Standardized date/time and location fields
* Designed and implemented relational schema in PostgreSQL with proper keys and constraints

---

### 📊 Data Analyst

* Developed business-driven SQL queries
* Performed joins, aggregations, and window functions
* Analyzed revenue, delivery performance, customer retention, and reviews

---

### 📈 Power BI Developer

* Built interactive dashboards for executive reporting
* Designed KPIs, slicers, and drill-down visuals
* Created data models to support performance tracking and insights

---

## 💼 Business Questions Answered

* Which sellers and product categories generate the highest and lowest revenue?
* How does revenue trend month-over-month and year-over-year?
* Which regions experience the longest delivery times?
* Are delayed deliveries associated with lower customer review scores?
* What is the repeat purchase rate and which cities have the most loyal customers?
* Which payment methods contribute the most to revenue?

---

## 📊 Key KPIs

* **Total Revenue**
* **Total Orders**
* **Total Items Sold**
* **Average Order Value (AOV)**
* **Late Delivery Rate**
* **Average Review Score**
* **Repeat Customer Rate**
